# Proofreading checklist
- [ ] Headings follow a single style (H1/H2…)
- [ ] Sentence length under control (avg ≤ 20 words)
- [ ] Passive voice minimized
- [ ] Numbers formatted consistently
- [ ] Acronyms expanded on first use
- [ ] Date/units style consistent
- [ ] Links live and descriptive
- [ ] Final spell/grammar pass
